MAC系统使用hexo配置博客

1 前期工作

注册github账户，创建个人仓库

2 在Hexo上创建blog

这一步的意思是现在你有了一个page，不过这个page只有本地链接才能打开，不具备web page的性质：即在世界各个角落的人输入同一个链接都能打开一样的页面。

2.1环境配置

  • 安装Node.js
  • 安装hexo

2.2 搭建blog

3 使用GitHub二级链接打开个人博客

这一步是上一步的延伸，能够达成把个人的github.pages展示给所有人，统一的链接是xxxxx.github.io，其中xxxxx表示的是你的个人仓库名，即用户名。

  • 安装Git

4 绑定自定义域名

进入blog文件夹：cd Desktop/blog2

hexo server

到目前为止，我们的本地博客就成功搭建了，但是现在我们只能通过本地连接查看博客，我们要做的是让其他人也能够访问我们的博客，这就需要我们将博客部署到Github Pages上

ssh-keygen -t rsa -C “your email@example.com“

ssh-keygen -t rsa -C “yueru26@126.com“

git config –global user.name “yueru914”

git config –global user.email “yueru26@126.com“

删除本地分支，重新拉取试试
